## Folding

Settings: `'foldmethod'`, `'foldmarker'`, `'foldexpr'`, `'foldlevel'`

Use this in `init.lua` to use Tree-Sitter for folding:

```lua
vim.o.foldmethod = "expr"
vim.o.foldexpr = "v:lua.vim.treesitter.foldexpr()"
```

Actions:

- `zc` / `zC` — Close fold (capital: recursive)
- `zo` / `zO` — Open fold (capital: recursive)
- `zr` / `zR` — Reduce folding, i.e. expand one level of folding (capital: open all folds)
- `zm` / `zM` — Increase folding, i.e. close one level of folding (capital: close all folds)
- `zv` — Open enough folds to reveal cursor
    - This one is useful if you search for some text with `/` and it happens to be inside a closed fold. Just press `zv` to reveal it.
- `zi` — Toggle folding
    - This enables/disables folding entirely. It’s useful to temporarily unfold everything while still preserving what was folded (or not) before.
