# Text object selection

`:h object-select`

Show:

- Around vs. in: `a<delim>`, `i<delim>`.
- Sentence: `s`
- Paragraph: `p`
- Surrounded blocks: `]`, `)`, `}`, `"`/`'`/```, `>`, `t`
    - Number prefix to traverse out of nested blocks.
- Indentation level: `i`
- Next search pattern match: `gn`
- Last selection: `gv`

# Indenting and wrapping text

`=` and `gq` actions.

# Commenting text

`gc` action.

# vim-surround

Change surrounding delimiters: `cs<delim><delim>`

Add new surrounding characters: `ys<motion><delim>` or visual mode then `S<delim>`.

Function call surrounding: (visual) `Sf`.

Custom surrounding: `vim.g.surround_42 = "/* \r */"`.

# Block visual mode

`C-v`

# vim-abolish

Case-preserving search-and-replace with `:S`.

# “Dumb” suggestions

`C-x` + ?.

See `:h ins-completion`.

# Autocommands

Useful events: `BufWritePost`, `FileType`
